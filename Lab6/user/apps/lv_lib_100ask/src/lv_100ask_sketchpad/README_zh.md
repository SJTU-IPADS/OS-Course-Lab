# 前言
**lv_100ask_sketchpad** 特性：

- 自定义画布样式
- 自定义画笔颜色
- 自定义画笔大小
- more todo...

`lv_100ask_sketchpad` 使用起来非常简单，后续自定义拓展功能也很方便，更多新功能敬请期待。
