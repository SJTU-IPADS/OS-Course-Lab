# Introduction
**lv_100ask_sketchpad** features：

- Custom canvas style
- Custom brush color
- Custom brush size
- more todo...

`lv_100ask_sketchpad` is very simple to use, and the subsequent custom expansion functions are also very convenient, so stay tuned for more functions.
