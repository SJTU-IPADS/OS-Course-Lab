# 前言
**lv_100ask_page_manager** 特性：

- 支持多级页面切换
- 跨页面切换
- 页面切换过渡动画
- 可以指定切换页面时是否删除当前页面(节省内存)
- 自定义页面过渡动画(TODO)
- 自定义页面返回按钮(TODO)
- 页面选择器(TODO)
- more todo...

`lv_100ask_page_manager` 使用起来非常简单，后续自定义拓展功能也很方便，更多新功能敬请期待。
